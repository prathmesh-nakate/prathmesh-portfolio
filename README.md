## 🚀 Getting Started

### Run Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/prathmesh-nakate/prathmesh-nakate.github.io.git
   ```

2. Open the folder and double-click `index.html`

   Or use the **Live Server** extension in VS Code for auto-refresh on save.

### Deploy on GitHub Pages

1. Push your code to a GitHub repository named `prathmesh-nakate.github.io`
2. Go to **Settings → Pages → Source → main branch / root**
3. Your site will be live at `https://prathmesh-nakate.github.io`

## ✏️ Customization

To personalize this portfolio:

1. Replace `Prathmesh` with your name throughout `index.html`
2. Update email, GitHub, and LinkedIn links in the Contact section
3. Add your real GitHub repo URLs to each project card
4. Replace `resume.pdf` with your actual resume file
5. Update the project descriptions to match your work
